# 简介

二维码扫码登录时，当用户完成扫码操作并授权登录后，开发者可以在应用中获取用户在飞书的身份信息，然后使用该身份信息完成登录的操作。本教程指导你如何快速接入二维码扫码登录，并实现获取用户身份信息。

## 流程简介

网页应用登录是基于 [OAuth 2.0](https://oauth.net/2/) 标准协议实现的，通过飞书账号授权登录第三方应用的能力。
