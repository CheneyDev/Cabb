# 简介

本教程介绍如何将已有的网页应用直接嵌入到飞书工作台，实现在飞书内一键接入已有的业务系统。

## 所需应用配置

本教程中，接入网页应用所需配置的能力如下所示。
| **配置项** | **所需能力**  |
| ------- | --------- |
| 应用能力    | 网页应用      |
| 网页应用配置  | 桌面端/移动端主页
