# 简介

当企业拥有多套审批系统时，可以选用飞书开放平台的审批服务，将三方审批系统的数据汇总至飞书中。无需跨平台、无需反复迁移数据，用户在飞书中即可统一完成审批信息处理，享受一站式的办公体验。本教程介绍如何通过开放平台审批服务的 OpenAPI 开发三方审批流程。
