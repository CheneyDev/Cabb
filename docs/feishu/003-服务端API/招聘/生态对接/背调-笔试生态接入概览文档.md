# 背调/笔试生态接入概览文档

## **概述**

飞书招聘生态对接旨在通过提供标准接入流程和开放业务事件与接口，允许在飞书招聘中集成和使用三方背调与笔试/测评服务

## **适用对象**

- 背调、笔试/测评的服务商，想为飞书招聘的客户提供服务；
- 有集成能力的飞书招聘客户，想在招聘系统中集成自研背调、笔试系统或集成未接入飞书招聘的三方服务商。
