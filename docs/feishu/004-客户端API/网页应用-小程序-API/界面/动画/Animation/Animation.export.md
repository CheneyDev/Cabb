# Animation.export

Animation.export() 用于导出动画队列。`export` 方法每次调用后，会清掉之前的动画操作。
每次 `export` 只会导出 **尚未被导出** 的动画组，如果某动画组已经被导出过，则会被清除。如果在调用 `export` 时存在尚未完成的 **动画组**，则未进入 **动画组** 的视觉变换不会生效（但也不会被删除，在下次调用 `step` 方法后会继续生效）。

## 支持说明

该接口仅支持小程序调用，对应的客户端版本支持情况如下所示。

## 输入
无

## 输出

名称 | 数据类型 | 描述
--- | --- | ---
actions | string | 动画队列。
