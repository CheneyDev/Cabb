# disableLeaveConfirm(Object object)

取消当前页面离开二次确认弹框。
**Notice**：说明：当前页面指的是调用tt.disableLeaveConfirm时**小程序栈顶页面**，与API在哪个页面实例调用无关，**tt.enableLeaveConfirm同理**。
