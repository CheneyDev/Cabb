# Beacon API 错误码
|错误码               |错误信息    |说明   |
|------------------|--------|-----|
|1716201             |unsupport|系统或设备不支持|
|1716202             |already start|已经开始搜索|
|1716203             |not startBeaconDiscovery|还未开始搜索|
|1716204             |location service unavailable|位置服务不可用 |
|1716205             |bluetooth service unavailable|蓝牙服务不可用|
