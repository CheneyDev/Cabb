# onMenuItemTap

## tt.onMenuItemTap(function fn)

监听工作台小组件菜单的用户点击。

## 输入

**function fn(Object param)**

| **属性**    | **类型**        | **说明**       |
| --------- | ------------- | ------------ |
| menuItem  | object        | 点击的菜单项的描述数据  |
|  ∟ key    | string        | 菜单项的唯一标示     |
| ∟ iconUrl | string        | 菜单项的图标的链接地址  |
| ∟ name    | i18n<string> | 菜单项的名称，支持国际化 |

## 示例

### 代码示例
```json
{
  	"tab": "api",
  	"item": "onMenuItemTap"
  }
```

```js
tt.onMenuItemTap(({ key, name, iconUrl }) => {
  console.log('菜单项', key, '被点击');
})
```

