# getContainerRect

获取容器尺寸。

## 输入
param 继承自[标准对象输入](https://open.feishu.cn/document/uAjLw4CM/uYjL24iN/block/api/standard-object-input)，无额外扩展属性。

## 输出

`success`返回对象的扩展属性：

名称 | 数据类型 | 描述
--- | --- | ---
width | number | 宽度，单位 px
height | number | 高度，单位 px
