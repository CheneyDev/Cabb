# onContainerResize

监听容器尺寸变化。

## 输入

| **属性**   | **类型** |**必填**| **说明**              |
| -------- | ------ | ------------------- | -------- |
| callback    | function | 是| 当容器尺寸变化时的回调函数  |          

## 输出
回调函数返回对象的属性：

名称 | 数据类型 | 描述
--- | --- | ---
width | number | 宽度，单位 px
height | number | 高度，单位 px
