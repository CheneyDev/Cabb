# setStorage

设置本地缓存数据。

## 输入

param 继承自[标准对象输入](https://open.feishu.cn/document/uAjLw4CM/uYjL24iN/block/api/standard-object-input)，扩展属性描述：

| **名称** | **数据类型** | **是否必填**   | **默认值** | **描述**  |
| ------ | ------ | --------- | ------ | ------- | ---------- |
| key    | string | 是        | -      | 键名      | PC 端 1.0.4 |
| data   | any    |否  | undefined      | 键名对应的数据 | PC 端 1.0.4 |

## 输出

各 callback 的参数均无额外扩展属性。
