# Record.applyTransaction
修改当前云文档小组件的 Record 数据，跟 setRecord 方法是等价的，只是通过 handler 的方式来产生 changesets，该方法为异步调用。

## 可用性说明

权限要求 | 视图可用说明 | 平台可用 | 场景
--- | --- | --- | ---
可写 | - 正文小组件  
- 全屏视图  
- 模态框视图  
- 弹窗视图 | - PC  
- 移动端 | \-

## 输入

Record 数据写入处理器
| **名称**  | **数据类型**         | **是否必填** | **描述**         |
| ------- | ---------------- | -------- | -------------- |
| handler | SetRecordHandler | 是        | Record 数据写入处理器 |
SetRecordHandler
```js
export type SetRecordHandler = (op: RecordOperation) => void;
```
