# ISVBlockData
小组件 Block 的数据结构，继承于 [BlockData](https://open.feishu.cn/document/uAjLw4CM/uYjL24iN/docs-add-on/05-api-doc/BlockData/blockdata)

| **名称**            | **数据类型** | **是否必填** | **描述**   |
| ----------------- | -------- | -------- | -------- |
| component_id      | string   | 是        | 小组件实例 id |
| component_type_id | string   | 是        | 小组件类型 id
