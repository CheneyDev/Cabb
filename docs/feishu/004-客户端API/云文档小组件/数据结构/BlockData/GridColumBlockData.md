# GridColumBlockData
分栏列 Block 的数据结构，继承于 [BlockData](https://open.feishu.cn/document/uAjLw4CM/uYjL24iN/docs-add-on/05-api-doc/BlockData/blockdata)

| **名称**      | **数据类型** | **是否必填** | **描述**                    |
| ----------- | -------- | -------- | ------------------------- |
| width_ratio | number   | 是        | 当前分栏列占整个分栏的比例，取值 [1, 99]
