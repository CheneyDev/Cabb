# TodoBlockData
任务列表 Block 的数据结构，继承于 [TextBlockData](https://open.feishu.cn/document/uAjLw4CM/uYjL24iN/docs-add-on/05-api-doc/BlockData/textblockdata/textblockdata)

| **名称** | **数据类型** | **是否必填** | **描述**        |
| ------ | -------- | -------- | ------------- |
| done   | boolean  | 否        | 该 todo 项是否已完成 |
| folded | boolean  | 否        | 折叠状态
