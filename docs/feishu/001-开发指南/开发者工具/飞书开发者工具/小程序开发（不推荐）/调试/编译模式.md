# 飞书开发者工具-编译模式
为了帮助开发者调试从不同场景值进入具体的页面，开发者可以添加或选择已有的自定义编译条件进行编译和代码预览。

> 注：编译条件跟项目相关，每个项目可以保存自己相关的编译条件
