# 功能配置

你可以通过组件功能配置接口来操作组件的表现。
**注意事项**：必须在[挂载文档成功](https://open.feishu.cn/document/uYjL24iN/uYDO3YjL2gzN24iN3cjN/event-listener/mount-success)后调用。

## 使用方式
```js
// 通过setFeatureConfig方法修改组件的配置属性
myComponent.setFeatureConfig({
    extensions: {
        like: {
            "disable": true // 隐藏点赞
        }
    }
});
```
