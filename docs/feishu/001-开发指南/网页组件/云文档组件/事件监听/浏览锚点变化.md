# 浏览锚点变化
## 示例
```js
// npm
myComponent.register(DocComponentEvent.CURR_ANCHOR, function(anchor) {
  // ...
});

// sdk
myComponent.register('CURR_ANCHOR', function(anchor) {
  // ...
});
```

## 返回
|属性|	类型|	说明|
| ---|----- | ------- | 
|anchor|	String |锚点 hash 值
