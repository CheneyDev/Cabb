# transform

## 介绍

CSS`transform`属性允许你旋转，缩放，倾斜或平移给定元素。这是通过修改CSS视觉格式化模型的坐标空间来实现的。

## 语法

```css
transform: translate(5px);

transform: translate(5px) scale(1.5);

transform: rotate(0.5turn);

transform: skew(30deg, 20deg);
```

## 标准化语法

```css
transform: <function>
```
