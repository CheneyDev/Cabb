# text-decoration

## 介绍

设置文本的修饰线外观。

## 语法

```css
text-decoration: none

text-decoration: underline

text-decoration: line-through
```

### 取值

-   `none`

**默认值。** 无修饰线。

-   `underline`

下划线。

-   `line-through`

贯穿线/删除线。

## 标准化语法

```css
text-decoration: none | underline | line-through
```
