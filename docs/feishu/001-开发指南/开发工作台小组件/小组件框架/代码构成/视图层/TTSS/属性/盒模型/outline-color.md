# outline-color

## 介绍

用于设置轮廓的颜色。

## 语法

```css
/* 轮廓是红色 */

outline-color: red;

/* 轮廓是绿色 */

outline-color: rgb(0, 255, 0);

/* 轮廓是蓝色 */

outline-color: #0000ff;
```

### 取值

-  `<color>`

### 标准化语法

```css
outline-color: <color>
```

## 例子

```css
outline-color: red;

outline-color: rgb(0, 255, 0);

outline-color: rgba(0, 255, 0, 0.3);

outline-color: #ff00ff;

outline-color: #aabbccdd;
```
