# font-size

## 介绍

设置文本文字大小。

## 语法

```css
/* <length> */
font-size: 12px;
font-size: 0.8em;
/* <percentage> */
font-size: 80%;
```

### 取值

-   `<length>`

-   `<percentage>`

## 例子

```css
<text> hello lynx, default font size </text>
<text style="font-size: 30px;"> hello lynx, font size 30px </text>
```
