# transition-duration

## 介绍

用于设置过渡动画执行持续的时间。

## 语法

```css
transition-duration: 1s;

transition-duration: 300ms;
```

### 取值

-   `<time>`

持续时间。

## 标准化语法

```css
transition-duration: <time>
```
