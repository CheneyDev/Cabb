# TTSS

TTSS 是一套样式语言，用于描述 TTML 的组件样式。

为了适应广大的前端开发者，从语法上来讲，TTSS 是 CSS 的严格子集，但是在部分特性中对其进行了扩充。
> 目前移动端 Native 渲染模式下，CSS 属性都不支持继承能力

## 已支持css列表

> 开发者书写的浏览器标准 css 属性，可能在 PC 环境下效果正常，但是在移动端Native 环境下不符合预期，因此为了确保各端效果一致，请提前参考以下已支持范围的列表
