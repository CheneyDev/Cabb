# 打开日历 
从飞书 3.40.0 版本开始支持。

## 使用场景
跳转并打开日历，打开界面为上一次离开日历时的视图。

## 协议
[https://applink.feishu.cn/client/calendar/open](https://applink.feishu.cn/client/calendar/open)

##  参数
无
