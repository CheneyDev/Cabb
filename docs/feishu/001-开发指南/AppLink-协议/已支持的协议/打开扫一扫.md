# 打开扫一扫功能
注意事项：
- 从飞书 5.7.0 版本开始支持
- PC端不支持
## 使用场景
通过applink 打开飞书扫一扫功能

## 协议
[https://applink.feishu.cn/client/qrcode/main](https://applink.feishu.cn/client/qrcode/main)
