# Android消息能力

|组件类型| 最低飞书版本 |生效基线|
|--|--------|--|
|能力组件| 7.1   |2.4.5|

## 组件说明
通过该能力，能够调用飞书的部分消息能力，比如，转发文件到飞书消息、获取飞书消息中的资源文件
