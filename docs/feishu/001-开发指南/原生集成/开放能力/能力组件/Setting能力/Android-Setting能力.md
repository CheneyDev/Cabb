# Android Setting能力

|组件类型| 最低飞书版本 |生效基线|
|--|--------|-|
|能力组件| 6.5  |2.2.3及以上版本|

## 组件说明
通过该API能力，可以获取飞书的动态setting配置，只在SaaS可用，私有化不支持。
