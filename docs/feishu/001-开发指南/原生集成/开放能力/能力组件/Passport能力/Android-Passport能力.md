# Android Passport能力

|组件类型| 最低飞书版本 |生效基线|
|--|--------|--|
|能力组件| 5.22  |2.0.2及以上版本|

## 组件说明
通过该API能力，可以调用飞书passport部分能力，比如获取用户登录状态、did等。
