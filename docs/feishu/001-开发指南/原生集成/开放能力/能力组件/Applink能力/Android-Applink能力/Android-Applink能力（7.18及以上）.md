# Android Applink能力（7.18及以上）

|组件类型|最低飞书版本|生效基线|
|--|--|--|
|能力组件|7.18|2.6.2及以上基线|

## 组件说明
通过该API能力，可以调用飞书的页面跳转、路由能力。
