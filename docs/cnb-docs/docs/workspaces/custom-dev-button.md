可根据需要自定义云原生开发启动按钮，如：按钮名称、按钮描述、按钮是否禁用、启动 cpu 核数。

在仓库中新增 `.cnb/settings.yml` 文件，配置内容详见 [UI 定制](../repo/settings.md)。
