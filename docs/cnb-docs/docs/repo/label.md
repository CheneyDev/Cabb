## 导入CNB公开仓库的标签

1. 进入标签管理页，点击“导入”（有标签时，按钮在页面右上角；没有标签时，按钮在页面正中）
2. 在弹框里粘贴 CNB 公开仓库的完整链接，点击“导入”

> 注：导入标签不会覆盖已有标签，增量导入
