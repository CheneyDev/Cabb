`云原生构建` 将成员分为五种角色：`访客`、`助手`、`开发者`、`管理员`、`负责人`。

对于**公开**仓库，非仓库用户和未登录用户也能访问，用户角色为`匿名`。

对于一些资源（PR、Issue、评论等），非仓库成员也可以创建，用户角色为`创建者`。

下表列出了各角色所能访问、操作的资源：
